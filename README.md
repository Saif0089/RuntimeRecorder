# RuntimeRecorder
 Runtime Recorder for "UnityCamera(s) + WebCam(s) + System Audio(s) + Microphone(s) Audio" using NatSuite's NatCorder plugins.

# Platforms
- Android
- iOS
- Windows
- OSX

# Thirt Party Plugins
- NativeGallery
- NatSuite's NatCorder
